# `penguin setup`

Initialize `~/.penguin/penguin.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
penguin setup
penguin setup --workspace ~/.penguin/workspace
```

To run the wizard via setup:

```bash
penguin setup --wizard
```
