# `penguin pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
penguin pairing list whatsapp
penguin pairing approve whatsapp <code> --notify
```
